## Overview

Projeto - Lastro Consignado is your centralized platform for creating, organizing, and managing project documentation. You can build comprehensive docs sites with MDX support, custom components, and seamless collaboration features. Designed for teams handling complex projects like consignado financing workflows, it streamlines documentation from initial setup to ongoing updates.

This platform empowers you to document APIs, guides, changelogs, and FAQs efficiently. Focus on content while it handles rendering, search, and navigation automatically.

## Quick Start

Get up and running in three steps.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at <code>https://dashboard.example.com</code> with your email.
  </Step>
  <Step title="New Project" icon="plus">
    Click "New Project" and select "Lastro Consignado Template".
  </Step>
  <Step title="Publish Docs" icon="globe">
    Edit <code>introduction.mdx</code> and hit publish.
  </Step>
</Steps>

## Brand Guidelines

<Expandable title="Using Brand Color #3B82F6" default-open="true">

Follow these guidelines to maintain consistency.

- **Primary Color:** `{`#3B82F6`}` for buttons, links, and accents.
- **Usage:** Apply to headings, icons, and call-to-action elements.

```css
:root {
  --brand-primary: #3B82F6;
}

.button {
  background-color: var(--brand-primary);
  color: white;
}
```

Avoid using it for text backgrounds to ensure readability.

</Expandable>
