## Prerequisites

Before you begin, ensure you have the following ready:

<Callout kind="info">

- A valid email address for account registration
- Access to a modern web browser like Chrome or Firefox
- Optional: Your `{API_KEY}` if using API endpoints (get one from `https://dashboard.example.com`)

</Callout>

## Sign Up for an Account

Follow these steps to create your Projeto - Lastro Consignado account quickly.

<Steps>
  <Step title="Visit the Dashboard" icon="monitor">
    Navigate to `https://dashboard.example.com` and click **Sign Up**.
  </Step>
  <Step title="Enter Your Details" icon="user">
    Provide your email, create a strong password, and agree to the terms.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for the verification link from `no-reply@example.com` and click it to activate your account.
  </Step>
  <Step title="Complete Onboarding" icon="check-circle">
    Log in and complete the short onboarding survey to unlock full features.
  </Step>
</Steps>

## Create Your First Documentation Space

Once logged in, set up a dedicated space for your project documents.

<Steps>
  <Step title="Click New Space" icon="plus">
    From the dashboard, select **New Documentation Space**.
  </Step>
  <Step title="Name Your Space" icon="edit-3">
    Enter a name like "Lastro Consignado Project" and add a brief description.
  </Step>
  <Step title="Set Permissions" icon="shield">
    Choose public, private, or team access. Invite collaborators via email.
  </Step>
  <Step title="Save and Enter" icon="arrow-right">
    Click **Create** to access your new space dashboard.
  </Step>
</Steps>

## Upload Initial Documents

Add your first files using the web interface or API.

<Tabs>
  <Tab title="Web UI" icon="upload">
    Drag and drop files directly into the space or use the **Upload** button.

    Supported formats: PDF, DOCX, MDX, images.

    <Image
      src="https://example.com/upload-ui.png"
      alt="Upload interface screenshot"
      width="800"
      height="400"
    />
  </Tab>
  <Tab title="API" icon="code">
    Use the Documents API to upload programmatically.

    <CodeGroup tabs="JavaScript,cURL">
      ````javascript
      const formData = new FormData();
      formData.append('file', documentFile);
      formData.append('spaceId', 'your-space-id');

      const response = await fetch('https://api.example.com/v1/documents', {
        method: 'POST',
        headers: {
          'Authorization': `Bearer ${YOUR_API_KEY}`
        },
        body: formData
      });
      ````

      ````bash
      curl -X POST https://api.example.com/v1/documents \
        -H "Authorization: Bearer YOUR_API_KEY" \
        -F "file=@/path/to/document.pdf" \
        -F "spaceId=your-space-id"
      ````
    </CodeGroup>

    <ParamField path="spaceId" param-type="string" required="true">
      ID of your documentation space.
    </ParamField>
  </Tab>
</Tabs>

## Set Up Your First Project

Configure a project within your space.

<Steps>
  <Step title="New Project" icon="folder-plus">
    Click **New Project** and name it "Consignado Workflow".
  </Step>
  <Step title="Add Pages" icon="file-text">
    Create pages like "Overview.mdx" and "API Reference.mdx".
  </Step>
  <Step title="Publish" icon="globe">
    Preview and publish to generate a shareable URL.
  </Step>
</Steps>
